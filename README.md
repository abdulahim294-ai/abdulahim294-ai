## 👋 مرحبا بك في عالم الأتمتة والتطوير الذكي!

أنا مطور ويب متكامل (Full Stack) ومهندس أتمتة ذكاء اصطناعي شغوف بابتكار الحلول الرقمية الذكية وبناء أنظمة برمجية تسهم في أتمتة أصعب العمليات وتوفير الوقت. أجمع بين مهارات برمجة المواقع الحديثة وتصميم البنى البرمجية المؤتمتة باستخدام أدوات الـ Low-Code والنماذج اللغوية الضخمة (LLMs).

- 🌍 **الموقع:** مقديشو، الصومال
- 💼 **التخصص الحالي:** تطوير الويب الشامل + أتمتة الذكاء الاصطناعي التطبيقي
- 📚 **التعلم المستمر:** الهندسة المتقدمة للأوامر (Advanced Prompt Engineering) والذكاء الاصطناعي التوليدي من كورس يرا بالشراكة مع جوجل

### 🔗 [بوتات أتمتة التليجرام الذكية](https://github.com/abdulahim294-ai?tab=repositories)
